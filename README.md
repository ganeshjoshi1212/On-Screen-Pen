# OnScreen Pen

A transparent on-screen annotation tool for Windows. Take handwritten notes over any lecture, presentation, or desktop content. Save notes slide-by-slide and export as a single PDF.

## 📥 Download

Get `OnScreenPen.exe` from the `dist/` folder — it's a single standalone file. No installation needed.

## 🚀 How to Use

1. **Double-click** `OnScreenPen.exe`
2. A **green circle (🖱️)** appears — this is the toolbar
3. Click it to expand the full toolbar
4. Switch to **Draw Mode** to start writing (press **F9** or click the button)
5. Draw your notes on screen
6. Click **📄 Add Slide to PDF** to save the current notes to memory and clear for the next page
7. Repeat for each topic / page of notes
8. When done, click **📕 Save & Export PDF** to combine all slides into one PDF document

## 🎨 Features

| Tool | Description |
|------|-------------|
| **Draw/Mouse Mode** | Toggle between drawing mode (blue) and click-through mode (green) where your desktop is fully usable. |
| **Pick Color** | Choose your pen color. |
| **Size** | Adjust the thickness of your pen. |
| **Undo** | Undo the last stroke you drew. |
| **Slide BG** | Choose a background color for your saved PDF slides (default: White or Black). |
| **Add Slide to PDF** | Captures the current annotations, clears the canvas, and stores the slide in memory. |
| **Clear Canvas** | Erase all strokes from the screen immediately. |
| **Save & Export PDF**| Saves all stored slides as one document on your computer. |

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| **F9** | Toggle Draw / Mouse mode |

*(Note: We removed extra keyboard shortcuts such as F10, F11, F12 to keep the software simple and non-intrusive!)*
